Global Installation

System-wide installation for production environments

NPM Global Install

bash 
# Install globally
npm install -g @farmfriend/ff-terminal

# Verify installation
ff --version

# Check installation path
which ff

System Requirements

  • Node.js: 20.0+ LTS
  • npm: 9.0+
  • Memory: 512MB minimum, 2GB recommended
  • Disk: 200MB for installation
  • OS: Linux (Ubuntu 20.04+), macOS 12+, Windows 10+

Unix/Linux Setup

bash 
# Add to PATH (if needed)
echo 'export PATH="$PATH:$(npm config get prefix)/bin"' >> ~/.bashrc
source ~/.bashrc

# Set permissions
sudo chown -R $(whoami) ~/.npm

# Verify
ff doctor

Windows Setup

powershell 
# Install globally
npm install -g @farmfriend/ff-terminal

# Verify
ff --version

# Add to PATH manually if needed
# npm config get prefix

Production Install Flags

bash 
# Production install (exact version)
npm install -g @farmfriend/ff-terminal@2.0.0

# Lock dependencies
npm config set save-exact true

# Ignore dev dependencies
npm install -g --production @farmfriend/ff-terminal
