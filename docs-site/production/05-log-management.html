Log Management

Logging, rotation, and monitoring

Log Levels

  • error - Critical errors requiring immediate attention
  • warn - Warning messages for potential issues
  • info - General information about operations
  • debug - Detailed debugging information

Environment Configuration

bash 
# Set log level
export FF_LOG_LEVEL=info

# Log file location
export FF_LOG_DIR=/var/log/ff-terminal

# Enable structured JSON logging
export FF_LOG_FORMAT=json

Log Rotation (logrotate)

bash 
# /etc/logrotate.d/ff-terminal
/var/log/ff-terminal/*.log {
    daily
    rotate 14
    compress
    delaycompress
    notifempty
    missingok
    create 0640 ff-user ff-user
    sharedscripts
    postrotate
        systemctl reload ff-terminal > /dev/null 2>&1 || true
    endscript
}

Viewing Logs

bash 
# Tail logs
tail -f /var/log/ff-terminal/daemon.log

# Search for errors
grep -i error /var/log/ff-terminal/daemon.log

# Systemd journal logs
journalctl -u ff-terminal -f
journalctl -u ff-terminal --since "1 hour ago"
