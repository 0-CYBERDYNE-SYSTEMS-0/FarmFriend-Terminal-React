Port Configuration

Network ports, firewall rules, and load balancing

Default Ports

Service Port Protocol Description
Daemon API 3000 HTTP Primary daemon endpoint
WebSocket 3001 WS Real-time events
Metrics 9090 HTTP Monitoring metrics

Environment Variables

bash 
# Daemon port
export FF_PORT=3000

# WebSocket port
export FF_WS_PORT=3001

# Metrics port
export FF_METRICS_PORT=9090

# Bind address (default: 0.0.0.0)
export FF_HOST=0.0.0.0

Firewall Rules (UFW)

bash 
# Allow daemon port (specific IPs only)
sudo ufw allow from 192.168.1.0/24 to any port 3000

# Allow WebSocket
sudo ufw allow from 192.168.1.0/24 to any port 3001

# Enable firewall
sudo ufw enable
sudo ufw status
