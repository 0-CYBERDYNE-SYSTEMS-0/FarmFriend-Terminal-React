Provider Abstraction

Unified interface for multiple AI providers

Overview

FF Terminal provides a unified abstraction layer that allows seamless switching between different AI providers. This enables users to choose the provider that best fits their needs.

Supported Providers

Provider Models Features
OpenAI GPT-4, GPT-4 Turbo, GPT-3.5 Streaming, function calling, vision
OpenRouter Claude, GPT-4, Llama, Mixtral Multiple models, routing
Google Gemini Gemini Pro, Gemini Ultra Multimodal, long context
Anthropic Claude 3 Opus, Sonnet, Haiku Large context, long outputs
Groq Llama, Mixtral, Gemma Fast inference
Local Models Ollama, LM Studio Private, offline capable

Provider Interface

typescript 
interface AIProvider {
  name: string;
  models: string[];
  
  complete(prompt: string, options: CompletionOptions): Promise;
  stream(prompt: string, options: CompletionOptions): AsyncIterator;
  chat(messages: Message[], options: CompletionOptions): Promise;
  countTokens(text: string): number;
}

interface CompletionOptions {
  model?: string;
  temperature?: number;
  maxTokens?: number;
  stream?: boolean;
}

Configuration

json 
{
  "provider": {
    "type": "openrouter",
    "apiKey": "${OPENROUTER_API_KEY}",
    "defaultModel": "anthropic/claude-3-opus",
    "fallbackModels": [
      "anthropic/claude-3-sonnet",
      "openai/gpt-4-turbo"
    ]
  }
}
