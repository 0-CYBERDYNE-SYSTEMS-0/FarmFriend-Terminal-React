Hooks System

Event-driven automation hooks

Available Hooks

Hook Trigger
before_tool Before tool execution
after_tool After tool execution
on_error When an error occurs
on_session_start Session is created
on_session_end Session ends

Configuration

json 
{
  "hooks": {
    "before_tool": [
      {
        "name": "log_tool_call",
        "command": "echo \"Tool: ${TOOL_NAME}\""
      }
    ],
    "on_error": [
      {
        "name": "notify_error",
        "command": "send_webhook ${ERROR_MESSAGE}"
      }
    ]
  }
}
