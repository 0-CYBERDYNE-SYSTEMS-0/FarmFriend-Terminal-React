FF Terminal

A dual-process AI agent with WebSocket-based terminal interface, web UI, and extensible skill system

🚀 Quick Start

Get up and running with FF Terminal in minutes:

bash 
# Clone the repository
git clone https://github.com/farmfriend/ff-terminal-ts.git
cd ff-terminal-ts

# Install dependencies
npm install

# Start the daemon (Terminal 1)
npm run daemon

# Start the CLI (Terminal 2)
npm run cli

✨ Key Features

  • Dual-Process Architecture: Separated daemon and UI for stability and flexibility
  • WebSocket Communication: Real-time, bidirectional messaging between CLI and daemon
  • Multiple Interfaces: Terminal CLI, Web UI, and FieldView Classic
  • Extensible Skill System: 40+ production-ready skills for common tasks
  • 35+ Built-in Tools: File operations, code search, AI integration, and more
  • Provider Abstraction: Support for OpenAI, OpenRouter, Gemini, and more
  • Session Management: JSON-based session persistence with UUID tracking
  • Workspace Isolation: Global and local workspace modes
  • AsyncLocalStorage Context: Distributed tracing across async boundaries
  • Multiple Personas: Agent personalities with different capabilities
