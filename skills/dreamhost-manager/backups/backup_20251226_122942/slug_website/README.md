
# Scrapmapple and the Fisbars Webpage

This project contains the files for a single-page website featuring a hero section with a background image inspired by the Pacific Northwest forests. The page plays music from SoundCloud at a low volume when opened.

## Files Included

- `index.html`: The main HTML file for the webpage.
- `styles.css`: The CSS file for styling the webpage.
- `script.js`: JavaScript file for handling the autoplay and volume control of the SoundCloud track.
- `background.jpg`: The background image for the hero section.

## Instructions

1. Upload all the files to your DreamHost domain.
2. Ensure that the file paths are correctly set in your hosting environment.
3. Open the webpage to see the hero section and hear the SoundCloud track autoplay at a low volume.
