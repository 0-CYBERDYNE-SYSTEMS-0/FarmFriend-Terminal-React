01 / Overview

Design Philosophy

Agrarian Brutalism is a design language that embraces raw, honest aesthetics rooted in agricultural utility. It combines the functional simplicity of brutalism with earth tones and organic textures found in farming environments. The style prioritizes clarity, utility, and an unpretentious approach to modern agricultural technology.

Core Principles

  • Raw & Honest: No decorative flourishes — every element serves a purpose
  • Functional First: Utility drives design decisions
  • Earth-Based: Color palette inspired by soil, rust, and harvest
  • Thick Borders: Brutalist 3-4px borders for structural definition
  • Offset Shadows: Shadow effects that create depth without softness
  • Clear Typography: Mix of display, heading, body, and mono fonts for hierarchy

09 / Responsive Design

Breakpoints

Mobile-first responsive design with breakpoints at key screen sizes. Use CSS Grid and Flexbox for layouts that adapt naturally to different screen sizes.

Breakpoint Size Usage
Mobile < 480px Single column, stacked layouts, smaller fonts
Tablet 481px - 768px 2-column grids, adjusted spacing
Desktop 769px - 1200px 3-4 column grids, full navigation
Large > 1200px Maximum content width (1400px), multi-column

Best Practices

  • Use clamp() for fluid typography
  • Use CSS Grid auto-fit for responsive card layouts
  • Max-width: 1400px for main container
  • Padding: var(--space-md) for side margins
  • Test on all breakpoints before deployment
