## Skill Management

### Loading Skills

```bash
# List all skills
/skills
Action: list

# Load specific skill
/skills
Action: load
Skill: remotion-expert

# Reload skill
/skills
Action: reload
Skill: shopify_ops
```

### Skill Triggers

Skills activate automatically based on keywords:

**Design triggers:**
- "design", "logo", "graphic", "illustration", "theme"

**Development triggers:**
- "component", "library", "web app", "swift", "package"

**Testing triggers:**
- "test", "debug", "review", "e2e"

- `name` - Skill identifier
- `description` - When to use this skill
- `tags` - Keywords for discovery
- `triggers` - Activation phrases

## Tips

1. **Use skill triggers** - Mention keywords to auto-activate skills
2. **Check skill metadata** - Each skill has description and tags
3. **Combine skills** - Use multiple skills in sequence for complex tasks
4. **Customize options** - Many skills accept configuration options
5. **Reload after updates** - Use skill reload to load latest changes
6. **Create custom skills** - Use skill-creator to build your own
7. **Document skills** - Each skill has SKILL.md with usage guide
8. **Version tracking** - Skills include version numbers for compatibility
9. **Skill dependencies** - Some skills require external tools or APIs
10. **Skill workflows** - Use skills-workflow tool for multi-skill orchestration
